WordCounter by pfwd

A Simple word counter plugin for jQuery which counts full words and not spaces.

A simple word counter plugin for jQuery. The plugin has the following features: Limits If a limit is supplied then the user will be warned if the word count exceeds the limit. Negative numbers can be turned on or off If the limit has been exceeded then you have the choice of showing a negative or positive word count. E.G words left: -10 or words to remove 10 Tailored messages to suit the application Each message can be changed. Messages settings: - defaultNoLimitFormat = the message shown if limit is null - limitExceededFormat = the message shown when a limit has been exceeded (Requires a limit) - limitNotExceededFormat = the message shown when a limit hasn't been exceeded (Requires a limit)
